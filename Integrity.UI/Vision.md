# Platform Vision
___
1. Context-backed UI with presence avatars and tab/window indicators
2. SystemTray backed Omnilauncher with keyboard shortcut
3. Multi-workspace context interchange with drag-and-drop object mapping.
4. Column-level record locks to allow multi-transient record changes
5. System notifications and alerts startup service to allow for notifications even when closed
6. Live socket connections for all substantive data and views.
